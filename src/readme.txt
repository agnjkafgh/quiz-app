Implement an online MCQ Test where
1.one question will appear at a time in front of the user
 with its four options, of which only one is correct.
2.Options should be jumbled each time.
3.Each correct answer will give you 10 points,
 and an incorrect will decrease 5 points
4.Whenever the user selects an option; the score should update.
5.The user should not be able to update the option once selected.
6.Each question should have `next` and `previous` buttons. 
7.User is allowed to go next only when an option is selected.
8.He can go back to view the previous question anytime.
 But should not be able to change previous responses




Questions API:- https://opentdb.com/api.php?amount=10
